# I_AILAB Paper Review
 인공지능 연구실의 Deep learning paper review study 입니다.  
매 주 2회 진행하는 Paper review 내용을 담고있습니다.

# Rule

- 최소 주 1회 발표
- 발표자료 폴더 이름과 presenter 테이블에는 논문의 풀네임 기입
- 발표자료 해당 폴더에 (Paper link, Key words) 기입
